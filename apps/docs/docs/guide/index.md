# 介绍

Agent Monitor 是一个现代化的智能体监控与客服系统，专为管理和优化您的 AI Agent 而设计。

## 核心功能

- **智能体监控**: 实时查看 Agent 状态、执行历史与错误日志。
- **智能客服**: 基于 Ollama 的 AI 自动回复，支持人工接管。
- **看板娘**: Live2D 虚拟助手，提供快捷菜单与互动功能。
- **用户管理**: 完善的权限控制与登录认证系统。

## 技术栈

- **前端**: React 18, Vite, Ant Design, Tailwind CSS, Zustand
- **后端**: Laravel 11, PHP 8.2, PostgreSQL, Redis
- **AI**: Ollama (支持多种开源大模型)
